JavaScriptのCanvas表示例

原点：画面の左上

X座標：画面の右向き

Y座標：画面の下向き

矩形（塗りつぶし）: fillRectメソッドを使って、指定した位置とサイズの矩形を塗りつぶしで描画します。
fillRectCtx.fillStyle = 'blue';
fillRectCtx.fillRect(50, 50, 300, 100);
矩形（枠のみ）: strokeRectメソッドを使って、指定した位置とサイズの矩形を枠だけで描画します。
strokeRectCtx.strokeStyle = 'black';
strokeRectCtx.strokeRect(50, 50, 300, 100);
円（塗りつぶし）: arcメソッドとfillメソッドを使って、指定した位置と半径の円を塗りつぶしで描画します。
パラメーター: context.arc(x, y, radius, startAngle, endAngle, anticlockwise)
x: 円の中心の x 座標, y: 円の中心の y 座標, radius: 円の半径
startAngle: 円弧の開始角度（ラジアンで指定）, endAngle: 円弧の終了角度（ラジアンで指定）, anticlockwise: 円弧を描く方向（省略可能）
circleCtx.beginPath();
circleCtx.arc(200, 100, 50, 0, Math.PI * 2, false);
circleCtx.fillStyle = 'red';
circleCtx.fill();
楕円（枠のみ）: ellipseメソッドとstrokeメソッドを使って、指定した位置とサイズの楕円を枠だけで描画します。
楕円は、内接する四角形（バウンディングボックス）を指定して描画します。内接する四角形が決まると、その4点を通る楕円は一つだけ決定されます。
パラメーター: context.ellipse(x, y, radiusX, radiusY, rotation, startAngle, endAngle, anticlockwise)
x: 楕円の中心の x 座標, y: 楕円の中心の y 座標, radiusX: 楕円の x 軸方向の半径, radiusY: 楕円の y 軸方向の半径
rotation: 楕円の回転角度（ラジアンで指定）, startAngle: 楕円弧の開始角度（ラジアンで指定）, endAngle: 楕円弧の終了角度（ラジアンで指定）, anticlockwise: 楕円弧を描く方向（省略可能）
ellipseCtx.beginPath();
ellipseCtx.ellipse(200, 100, 100, 50, 0, 0, Math.PI * 2);
ellipseCtx.strokeStyle = 'green';
ellipseCtx.stroke();
直線: moveToメソッドとlineToメソッドを使って、指定した開始点から終了点までの直線を描画します。
lineCtx.beginPath();
lineCtx.moveTo(50, 150);
lineCtx.lineTo(350, 50);
lineCtx.strokeStyle = 'purple';
lineCtx.lineWidth = 5;
lineCtx.stroke();
